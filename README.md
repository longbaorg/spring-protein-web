### VUE管理系统

---

此项目是vue+element-ui 快速开发的物资管理系统，后台用的java springBoot 所有数据都是从服务器实时获取的数据，具有登陆，注册，对数据进行管理，打印数据等功能

### 项目运行

---



拷贝后

~~~javascript
npm install

npm run dev
~~~

### 注意
 在项目根目录新建postcss.config.js文件，并对postcss进行配置 有的话就不用了
 添加下面代码  就可以正常运行了
 
 ~~~javascript
 module.exports = { 
  plugins: { 
    'autoprefixer': {browsers: 'last 5 version'} 
  } 
}
~~~
